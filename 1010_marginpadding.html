박스속성_margin/padding

웹페이지의 레이아웃을 구성할 때 가장 중요한 요소
left/ right/ top/ bottom 으로 낱개로 설정 가능.

작성법

  • 한가지 숫자 적용 - margin:10px 작성 시 4면 모두 동시 적용 (=패딩)
  • 두가지 숫자 적용 - margin:10px 20px 작성 시 상하:10px 좌우:20px 적용
  • 네가지 숫자 적용 - margin:5px 10px 15px 20px 작성 시 상 우 하 좌 적용 (위에서부터 시계방향으로)
  • 건바이건으로 적용 - margin-bottom:10px

margin이란?

테두리와 다른 태그의 간격을 설정

  • 요소1/margin-bottom:10px
  • 요소2/margin-top:20px

padding이란?

요소의 내부여백을 설정하여 공간으로 확보하는 형태

  • 요소1
  • 요소2

width / height

가로값/높이값으로 지정할 수 있으며 padding과 border값의 영향을 받음

  • Lorem ipsum dolor sit amet consectetur adipisicing elit.
  • 요소2
  • Lorem ipsum dolor sit amet consectetur adipisicing elit.
  • 요소2

border란?

테두리의 모양, 색깔, 두께를 나타냄

  • 전체 테두리: 5px 검정색 실선
  • 왼쪽 선: 20px 빨간색 실선
  • 위쪽 선: 1px 파란색 도트선
  • 오른쪽 선: 10px 파란색 두줄 선
  • 아래쪽 선: 2px 보라색 긴 점선
