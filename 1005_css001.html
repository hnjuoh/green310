CSS란?

Cascating Style Sheet 약자로 웹페이지를 꾸미려고 작성하는 코드.
내부 스타일시트, 외부 스타일시트, 인라인 스타일시트 등으로 세가지 종류가 있으며
인라인 스타일 시트는 태그 내부에 직접 기재하는 방식으로 우선 적용되므로
많은 내용보다는 가장 중요한 한 두문장을 표현한다.

  • 내부시트 적용 - style 태그 안에 코드를 작성하며 body와 head 어느 공간에 있더라도 적용됨
  • 외부시트 적용 - HTML 태그 내부에 link태그로 주소 연결하거나 스타일 시트 내부에 @import로 주소 연결을 하기도 함.
  • 인라인시트적용 - 태그 내부에 속성값으로 작성함.

선택자

선택자명 설명
* 전체태그 선택 / 다수의 태그를 선택할 때 ,를 사용하는데 html내부에서 사용한 모든 태그를 부를 때 사용함
타입선택자 태그이름을 그대로 가져다 쓰는 형태
id선택자_ # id가 붙어있는 태그를 부를 때 사용하는 형태
클래스선택자_. 클래스가 붙어있는 태그를 부를 때 사용하는 형태 클래스 작성 뒤에 한칸 띄어쓰기 후 뒤에 있는 클래스만 불러서도 사용가능
자식 선택자 선택자>선택자 or 선택자 선택자 현재 요소 하위에 들어가있는 요소를 부를때 자손자 방식/후손자방식 사용가능
종속선택자_ 결합형 선택자.클래스명 or 선택자#클래스명 태그 이름중에 클래스 혹은 아이디 이름을 가지고 있는 요소만 특정지어 부르는 형태
세부클래스
  • first-letter - 요소의 첫글자
  • :first-child/ last-child - 요소형제 중 첫번째 혹은 마지막 요소를 선택
  • :before / :after - 요소의 맨앞 혹은 맨뒤 추가적으로 HTML에 없는 요소를 만들어내는 가상 요소
